Pour installer le Plugin depuis GitHub : 

1. télécharger l'intégralité des fichiers et des dossiers liés EN GARDANT LA MÊME ARBORESCENCE
OU
1. télécharger le ZIP des fichiers de code "topeau-master.zip"

2. construire un dossier "topeau" contenant l'intégralité des fichiers et dossiers EN GARDANT L'ARBORESCENCE
OU
2. dézipper le dossier "topeau-master" et renommer en "topeau"

3. déposer le dossier "topeau" en suivant ce chemin : 

> Disque local C:
	> Utilisateurs
		> sélection du dossier utilisateur
			> AppData (si caché : chercher dans la barre de recherche d'emplacement)
				> Roaming
					> QGIS > QGIS3 > profiles > default
						> python
							> plugins
								> déposer dossier topeau avec les autres dossiers plugins


4. si QGIS ouvert : fermer la fenêtre et relancer
